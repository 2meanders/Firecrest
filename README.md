# Firecrest

Firecrest is a modern C++ rendering library designed to be customizable and extensible. It provides an intuitive interface for creating interactive graphical applications while maintaining the flexibility for advanced users to extend and customize its functionality.

## Overview

The library's architecture focuses on providing components that can be easily combined to create complex UI layouts and rendering effects.

## Key Design Principles

- **Intuitive**: Simple things should be simple, complex things should be possible
- **Extensible**: Easy to add new components and customize existing ones
- **Modern**: Built with modern C++ practices and standards

## Building

Firecrest uses CMake as its build system.
